# 📖 Como Configurar GitHub Pages - Guia Completo

## Passo 1: Abra seu Repositório no GitHub

1. Acesse: https://github.com/clinicamayarecepcao-design/CALCULAREXAME
2. Você verá a página principal do repositório

```
┌─────────────────────────────────────────────┐
│  ⭐ clinicamayarecepcao-design/CALCULAREXAME │
│                                              │
│  Code  Issues  Pull requests  Discussions  │
│  ⬇️                                          │
│ [Settings] ← Clique aqui (abas principais)  │
└─────────────────────────────────────────────┘
```

---

## Passo 2: Clique em "Settings"

1. Procure pela aba **Settings** (fica nas abas principais do repositório)
2. Clique nela

```
Menu superior do repositório:
┌──────────────────────────────────────────────┐
│ Code | Issues | Pull | Discussions | Settings│
│                                              │
│                                ↑ CLIQUE AQUI │
└──────────────────────────────────────────────┘
```

---

## Passo 3: Vá para "Pages"

1. Na página Settings, procure na **barra lateral esquerda**
2. Scroll down até encontrar **"Pages"**
3. Clique em "Pages"

```
Barra lateral esquerda:
┌──────────────────────┐
│ General              │
│ Access               │
│ Moderation options   │
│ Code and automation  │
│ ├─ Actions           │
│ ├─ Hooks             │
│ │                    │
│ Pages ← CLIQUE AQUI  │
│                      │
└──────────────────────┘
```

---

## Passo 4: Configure o Deploy

Depois de clicar em "Pages", você verá:

```
┌─────────────────────────────────────────┐
│ GitHub Pages                             │
│                                          │
│ Build and deployment                    │
│                                          │
│ Source:                                  │
│ ┌────────────────────────────────────┐  │
│ │ ⬇️ Deploy from a branch             │  │
│ └────────────────────────────────────┘  │
│                                          │
│ Branch:                                  │
│ ┌────────────────────────────────────┐  │
│ │ ⬇️ main     /  calculadora-exames  │  │
│ │              ↑ Selecione a pasta   │  │
│ └────────────────────────────────────┘  │
└─────────────────────────────────────────┘
```

---

## Passo 5: Selecione a pasta correta

1. **Primeira caixa**: Deixe como **Deploy from a branch** (padrão)

2. **Segunda caixa (Branch)**: 
   - Clique no dropdown
   - Selecione **"main"**

3. **Terceira caixa (folder)**: 
   - Clique no dropdown (provavelmente mostra "/ (root)")
   - Selecione **"/ calculadora-exames"**

```
                    ⬇️ Click aqui
         ┌──────────────────────────┐
         │ / (root)                 │
         │ / calculadora-exames ✓   │ ← Selecione esta
         └──────────────────────────┘
```

---

## Passo 6: Clique em "Save"

```
┌──────────────────┐
│ [Save] ← CLIQUE! │
└──────────────────┘
```

**Aguarde 1-2 minutos** para o GitHub processar.

---

## Passo 7: Seu site está ONLINE! 🎉

Após alguns segundos, você verá uma mensagem verde:

```
┌─────────────────────────────────────────────────┐
│ ✅ Your site is ready to be published at:       │
│    https://clinicamayarecepcao-design.github.io/│
│    CALCULAREXAME/calculadora-exames/             │
│                                                 │
│ [Visit site] ← Clique para abrir               │
└─────────────────────────────────────────────────┘
```

---

## 🎯 Link Final

Seu calculadora estará disponível em:

**https://clinicamayarecepcao-design.github.io/CALCULAREXAME/calculadora-exames/**

---

## ❌ Se não funcionar, verifique:

1. ✅ Você selecionou a pasta **"calculadora-exames"**?
2. ✅ Clicou em "Save"?
3. ✅ Aguardou 2-3 minutos?
4. ✅ A pasta tem: `index.html`, `script.js`, `style.css`, `exames.json`?

Se ainda não funcionar, envie uma mensagem com a mensagem de erro que aparece!
